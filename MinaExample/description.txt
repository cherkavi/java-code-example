Данный Framework предназначен для удаленного получения данных
от клиентов
То есть поднимается слушатель запросов и начинается диалог с удаленной стороной.
( при чем слушатель получает данные от того кто шлет, контроллирует что ему шлют,
а затем посылает ответ )

проверка текущего соединения:

telnet 127.0.0.1 9123
# любой ввод символов возвращает время

# quit - завершение сессии
