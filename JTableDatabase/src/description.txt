Данный пакет позволяет получать таблицу ( JTable) 
на основании данных из таблицы базы данных.
Есть запрос с кодами, а так же есть запрос с данными.
Запрос с кодами - используется для получения общего
числа данных и кодов, которые отображаются в данный 
момент.
Запрос с данными - запрос, на основании которого
можно получить все интересующие данные, подставив
нужный код
